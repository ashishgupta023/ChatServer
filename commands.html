How to use the TCP client

/rooms

Shows all the currently available chatroom names. 5 chat rooms are available by default by the chat server.

Example : /rooms

/join [room-name]

Use this to join a particular room. Room name is mandatory. No spaces allowed. A user in a chat room will receive messages from all users which includes the user in his current chat room , however the messages send by him will only be send to users in the current chat room.

Example: /join chat

/addroom [room-name]

Use this to add a new chat room. Room name is mandatory. No spaces allowed.

Example: /addroom mordor

/private [user-name]

If you are in a chat room, you can request a private chat with one of the members of the chat room. User name is mandatory. Private chat is not available outside chat room. If the request is not accepted within 20 seconds, the request is timed out. Person in a private chat will receive messages only from the user in the private chat dialogue.

Example: /private gandalf

/accept [user-name]

The user receives a notification when somebody requests a private chat with them. Use this command to accept the invitation to go in a private chat dialogue.User name is mandatory. Person in a private chat will receive messages only from the user in the private chat dialogue.

Example: /accept sauron

/leave

Use this to leave a chat room or when you are in a private conversation. If you are in a private conversation, this ends the private chat and takes you back to the room you were in. If you are in a chat room , takes you out of the chat room .

Example: /leave

/quit

Leave the chat !!

Example: /quit
